# FosholBari Server

Express 5 + MongoDB + better-auth backend for the FosholBari agricultural marketplace.

## Stack

- **Runtime:** Node.js, TypeScript
- **Framework:** Express 5
- **Database:** MongoDB 7 (native driver)
- **Auth:** better-auth (email/password + Google OAuth, bearer + JWT)
- **Dev runner:** `ts-node-dev` (hot reload)

## Setup

```bash
npm install
```

Create `.env` from the template below.

### Environment Variables

```
PORT=11111
MONGODB_URI=mongodb+srv://...
BETTER_AUTH_SECRET=...
BETTER_AUTH_URL=http://localhost:11111
CLIENT_ID=...             # Google OAuth
CLIENT_SECRET=...         # Google OAuth
CLIENT_URL=http://localhost:3000
```

## Commands

| Action | Command |
|--------|---------|
| Dev (hot reload) | `npm run dev` |
| Build | `npm run build` |

## API Routes

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/` | No | Health check |
| GET | `/explore` | No | Public listing (paginated, filterable) |
| GET | `/explore/filters` | No | Distinct categories & tags |
| GET | `/explore/:id` | No | Single product |
| POST | `/explore` | Any | Create product |
| PATCH | `/explore/:id` | Farmer | Update own product |
| DELETE | `/explore/:id` | Farmer | Delete own product |
| GET | `/orders?role=buyer|farmer` | Any | List own orders |
| POST | `/orders` | Buyer | Place order |
| PATCH | `/orders/:id` | Farmer/Buyer | Update order status |
| GET | `/demands?my=true` | Any | List crop demands |
| POST | `/demands` | Any | Create demand |
| POST | `/demands/:id/comments` | Farmer | Comment on demand |
| GET | `/bazar-notes` | Any | List shopping lists |
| POST | `/bazar-notes` | Any | Create note |
| PATCH | `/bazar-notes/:id` | Any | Update note |
| DELETE | `/bazar-notes/:id` | Any | Delete note |
| GET | `/profile` | Any | Get extended profile |
| PATCH | `/profile` | Any | Upsert profile |
| GET | `/stats/farmer` | Farmer | Farmer dashboard stats |
| GET | `/stats/buyer` | Buyer | Buyer dashboard stats |
| GET | `/stats/admin` | No | Global stats |
| GET | `/admin/users` | Admin | List users |
| PATCH | `/admin/users/:id` | Admin | Update user role |
| DELETE | `/admin/users/:id` | Admin | Delete user |
| GET | `/admin/orders` | Admin | List all orders |
| PATCH | `/admin/orders/:id` | Admin | Update any order |
| DELETE | `/admin/orders/:id` | Admin | Delete order |

Auth endpoints are auto-generated by better-auth at `/api/auth/*`.

## Vercel Deploy

The `vercel.json` deploys `index.ts` as a serverless function via `@vercel/node`. Set all environment variables in the Vercel dashboard.
